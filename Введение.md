# Введение

## Ввод информации

> `scanf(форматирующая_строка, переменные...)`
> Вводит информацию в переменные

Пример
```c
int a, b;
scanf("%d %d", &a, &b);
```

### Спецификаторы типов

`%с` - char\
`%d` - int\
`%f` - float\
`%s` - string

### Форматирующая строка

```c
%[спецификатор][длина][.точность]тип
```
