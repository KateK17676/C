# Математические операции

> `+`, `-`, `*`, `/`, `%`

> **Закон деления**
> Если целое число делится на целое число, то результат будет целочисленным.

## Обычное деление

```c
int a = 9;
int b = 5;
printf(a / (float)b) // 1.8
printf(a / b * 1.0) // 1.8
```

## Целочисленное деление

```c
int a = 9;
int b = 5;
printf(a / b) // 1
```

## Остаток от деления

```c
int a = 9;
int b = 5;
printf(a % b) // 4
```
